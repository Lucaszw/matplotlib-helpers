Grid plot
---------

The :func:`matplotlib_helpers.chart.encode` function is inspired by the
`altair <https://github.com/ellisonbg/altair>`__ project.

With :func:`matplotlib_helpers.chart.encode` **(quoted from the**
`altair <https://github.com/ellisonbg/altair>`__ **documentation)**:

    -  The **data source** is a ``DataFrame`` that consists of columns
       of different data types (quantitative, ordinal, nominal and
       date/time).
    -  The ``DataFrame`` is in a `tidy
       format <http://vita.had.co.nz/papers/tidy-data.pdf>`__ where the
       rows correspond to samples and the columns correspond the
       observed variables.
    -  The data is mapped to the **visual properties** (position, color,
       size, shape, faceting, etc.) using the group-by operation of
       Pandas.
